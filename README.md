# Newton's Third Law Stations Simulation

Interactive simulation for **Station 1** (collision force graphs) and **Station 4** (pushing-force scenarios) from the Newton's Third Law activity.

## Includes

- Presets for:
  - Collision 1
  - Collision 2 (heavier Cart B)
  - Equal push (no motion)
  - Push together
  - Cart B at wall
  - Sensor setup challenge
- Live force-vs-time graphs for both carts:
  - Force on A by B
  - Force on B by A
- Run comparison table (peak force, impulse, and speed change)
- Force diagram builder with `On ___ by ___` labels
- Dark/Light theme toggle using The Thinking Experiment defaults

## Files

- `index.html` UI structure
- `styles.css` theme + responsive layout
- `src/main.js` simulation engine, graphing, and diagram tools
